## Wi‑Fi Calling 开启

### iPhone

打开美国代理/VPN，→ 连接 Wi‑Fi →【设置】→【蜂窝网络】→【Wi‑Fi 通话】→ 打开【在 iPhone 上启用 Wi‑Fi 通话】，首次启用需填写美国 E911 地址（可以打开 Google 地图生成地址）。

### Android

打开美国代理/VPN，→ 连接 Wi‑Fi →【设置】→【连接/网络】→【Wi‑Fi 通话】→ 选择【优先 Wi‑Fi】，首次启用需填写美国 E911 地址（可以打开 Google 地图生成地址）。
